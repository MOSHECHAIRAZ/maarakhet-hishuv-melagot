# מערכת חישוב מלגות - גרסה עצמאית

מערכת מלגות עצמאית בפונקציונליות מלאה, לא תלויה במערכות אחרות.

## התקנה והפעלה

### דרישות מקדימות
- Node.js (v14 ומעלה)
- npm

### שלבים

1. **התקנת תלויות:**
```bash
npm install
```

2. **הפעלת השרת:**
```bash
npm start
```

השרת יפעל ב-`http://localhost:3001`

3. **פתחו את המערכת:**
פתחו את הדפדפן וכנסו לכתובת:
```
http://localhost:3001
```

## מבנה התיקייה

```
מערכת חישוב מלגות/
├── index.html              # דף הבית של המערכת
├── server.js               # שרת Express עצמאי
├── package.json            # תלויות Node.js
├── README.md               # קובץ זה
├── data/
│   └── scholarships.json   # מסד נתונים של המלגות
├── קובצי CSV              # קבצי נוכחות להעלאה
└── דוחות PDF              # דוחות שנוצרו
```

## תכונות

✅ **חישוב מלגות אוטומטי** - על בסיס נתוני נוכחות
✅ **היסטוריית חישובים** - שמירה וטעינה של חישובים קודמים
✅ **דוחות** - הפקת דוח כללי ודוחות אישיים
✅ **ייצוא** - ייצוא לקובץ CSV
✅ **עצמאי** - אין תלות במערכות אחרות

## השתמש

1. **העלאת קובץ נוכחות:**
   - גרור קובץ CSV לאזור הטעינה או לחץ לבחירה

2. **הגדרת פרמטרים:**
   - סכום מלגה חודשית
   - ימי לימוד בחודש
   - תקופות זמנים (אופציונלי)

3. **חישוב:**
   - לחץ על כפתור "חשב מלגות"

4. **שמירה והדפסה:**
   - שמור חישוב להיסטוריה
   - הפק דוחות או ייצא לקובץ

## מבנה קובץ CSV

הקובץ צריך להכיל עמודות:
- תעודת זהות
- שם פרטי
- שם משפחה
- שעת כניסה (HH:MM)
- שעת יציאה (HH:MM)

## API Endpoints

### שמירת חישוב
```
POST /api/scholarships
```

### קבלת כל החישובים
```
GET /api/scholarships
```

### קבלת חישוב ספציפי
```
GET /api/scholarships/:id
```
