# מערכת חישוב מלגות - גרסה עצמאית

מערכת מלגות עצמאית בפונקציונליות מלאה, לא תלויה במערכות אחרות.

[![GitHub](https://img.shields.io/badge/GitHub-Repository-blue)](https://github.com/MOSHECHAIRAZ/scholarship-system)

## התקנה והפעלה

### דרישות מקדימות
- Node.js (v14 ומעלה)
- npm

### שלבים

1. **התקנת תלויות:**
```bash
npm install
```

2. **הפעלת השרת:**
```bash
npm start
```
או השתמש ב-`start.bat` (ל-Windows) - הוא יפתח את הדפדפן אוטומטית.

השרת יפעל ב-`http://localhost:3001`

## תכונות

✅ **חישוב מלגות אוטומטי** - על בסיס נתוני נוכחות מקבצי CSV
✅ **היסטוריית חישובים** - שמירה וטעינה של חישובים קודמים
✅ **דוחות** - הפקת דוח כללי ודוחות אישיים (PDF)
✅ **ייצוא** - ייצוא לקובץ CSV
✅ **עצמאי** - אין תלות במערכות אחרות
✅ **ממשק ווב** - קל לשימוש דרך דפדפן

## השימוש במערכת

1. **העלאת קובץ נוכחות:**
   - גרור קובץ CSV לאזור הטעינה או לחץ לבחירה

2. **הגדרת פרמטרים:**
   - סכום מלגה חודשית
   - ימי לימוד בחודש
   - תקופות זמנים (אופציונלי)

3. **חישוב:**
   - לחץ על כפתור "חשב מלגות"

4. **שמירה והדפסה:**
   - שמור חישוב להיסטוריה
   - הפק דוחות או ייצא לקובץ

## מבנה קובץ CSV

הקובץ צריך להכיל עמודות:
- תעודת זהות
- שם פרטי
- שם משפחה
- שעת כניסה (HH:MM)
- שעת יציאה (HH:MM)

## API Endpoints

### שמירת חישוב
```
POST /api/scholarships
```

### קבלת כל החישובים
```
GET /api/scholarships
```

### קבלת חישוב ספציפי
```
GET /api/scholarships/:id
```
