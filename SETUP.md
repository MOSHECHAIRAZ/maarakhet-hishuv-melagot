# 🚀 התחלה מהירה - מערכת חישוב מלגות

## 3 שלבים בלבד!

### 1️⃣ התקנה
```bash
npm install
```

### 2️⃣ הפעלה
```bash
npm start
```
או לחץ על `start.bat` (בחלונות)

### 3️⃣ פתיחה בדפדפן
```
http://localhost:3001
```

---

## זהו! ✅

המערכת תפעל בנפרד מלא ממערכות אחרות.

### מה בתוך התיקייה?

| קובץ | תיאור |
|------|-------|
| `index.html` | ממשק המשתמש |
| `server.js` | שרת עצמאי (Express) |
| `package.json` | תלויות ותצורה |
| `data/` | מסד נתונים מקומי |
| `start.bat` | דרך קלה להתחלה (Windows) |
| `README.md` | תיעוד מלא |

### שימוש

1. העלה קובץ CSV עם נתוני נוכחות
2. הגדר פרמטרים (מלגה, ימי לימוד)
3. לחץ "חשב מלגות"
4. שמור או הדפס דוחות

### נתונים

כל הנתונים משמרים בתיקייה `data/` בקובץ JSON ריק בשרת המקומי.
לא כל תלויות ממערכות אחרות!
