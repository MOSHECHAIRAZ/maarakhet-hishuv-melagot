# 🚀 START - מערכת חישוב מלגות (עבור מתחילים בלבד!)

> **⏰ זה ייקח 3 דקות בלבד!**

---

## שלב 1️⃣: פתחו Terminal/CMD

### Windows:
```
1. לחץ Windows + R
2. הקלד: cmd
3. לחץ Enter
```

### Mac:
```
1. Cmd + Space
2. הקלד: Terminal
3. לחץ Enter
```

### Linux:
```
פתח Terminal
```

---

## שלב 2️⃣: עברו לתיקייה

```bash
cd "C:\Users\<שם_משתמש>\Projects\Desktop\אפליקציות ותוכנות\מערכת חישוב מלגות"
```

**או בפשוטות:**
```bash
cd "מערכת חישוב מלגות"
```

---

## שלב 3️⃣: התקנת הכלים (פעם ראשונה בלבד!)

```bash
npm install
```

**זה יטעון את Express (השרת)**

---

## שלב 4️⃣: הפעלה

```bash
npm start
```

**תראו:**
```
✅ Scholarship Calculator Server running on port 3001
```

---

## שלב 5️⃣: פתחו בדפדפן

**כתובו בשורת הכתובת:**
```
http://localhost:3001
```
